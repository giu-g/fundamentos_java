O que é Pacote?

Quando desenvolvemos um sistema OO, definimos um conjunto de classes. Para organizar essas classes surge o conceito de pacote.

Pacote é um envoltório de classes, ou seja, guarda classes e outros pacotes logicamente semelhantes ao pacote que os contém.

Podemos visualizar os pacotes como diretórios ou pastas, nosquais podemos guardar arquivos (classes) e outros diretórios (pacotes) que tenham conteúdo relacionado com o pacote que os contém.

Como Implementar Classes em Pacotes?

Para indicar que uma classe Java pertence a um determinado pacote, a primeira linha de código deve ser a declaração de pacote implementada através da diretiva package, seguida do nome do pacote, obedecendo a seguinte sintaxe:

package nomedopacote;

Essencialmente, uma classe denominada MinhaClasse que pertence a um pacote nomedopacote tem um “nome completo” que é nomedopacote.MinhaClasse. Assim, se outra aplicação tiver uma classe de mesmo nome, não haverá conflitos, pois classes em pacotes diferentes têm nomes completos distintos.

Como Implementar Classes em Pacotes No Java?

1) Com o cursor em cima do pacote, clique com o botão direito do mouse.

2) Selecione a opção "New".

3) Clique em "Class".

4) Configure sua classe.
